This repository contains structured and well-documented solutions to fundamental C programming concepts including:

- Matrix multiplication
- Structures & functions
- String handling
- Sorting using qsort
- Bitwise operations
- Number system conversions (Binary, Octal, Hexadecimal)

All programs are written in clean, exam-oriented and viva-ready format.
